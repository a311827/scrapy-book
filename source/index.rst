Welcome to ZenLive User Guide
==========================

Zen Cloud Live Service(ZenLive) provides you with high-definition smooth, low-latency, high-concurrency audio and video cloud processing services. Including core functions such as real-time transcoding, slice storage, and distribution acceleration. Bring a smooth access experience to end users and help the video live broadcast service go online quickly.
